Research

Hunting exoplanets with Machine Learning

Transiting exoplanet surveys generate a large number of data in the form of lightcurves. Current algorithms are able to work with big datasets to identify potential exoplanet candidates. However, these lists contain many false positives and further analyses is required ... to narrow down the list to the most likely candidates. Machine Learning (ML) is a wonderful tool to classify new data when there are previous datasets available. It has been shown that ML is able to correctly classify targets from the Kepler Objects of Interest (KOI) with a 90% accuracy. In my work I employ similar ML methods to classify planets from ground-based surveys such as SuperWASP. Read More


Characterizing exoplanetary atmospheres with ARIEL

This project addresses current topics in field of atmospheric characterization and aims to highlight exoplanetary science in the ARIEL era. Atmospheric retrieval is an inverse method used to recover the properties of exoatmospheres ... from its transmission or emission spec- trum. Retrievals from simulated spectra are analysed to determine the capabilities of ARIEL. Synthetic data sets are generated with state-of-the-art code to estimate the noise on ARIEL observations. Thence, I combine the data sets with forward models to perform the retrievals of the physical and chemical properties of the atmosphere. Read More
